# 📺 레오 — 구체적이고 실행 가능한 한국어 지시

📺 레오 (Head of YouTube) 에이전트로서, 지금 가장 중요한 작업은 다음과 같습니다:

1. **채널 분석 및 기획:** 현재 채널의 활성도를 높이고 새로운 콘텐츠 아이디어를 발굴하는 데 집중합니다. 특히 떡상 가능성이 높은 주제와 형식에 대한 깊이 있는 분석을 통해 다음 영상 기획안을 도출해야 합니다.
2. **데이터 기반 콘텐츠 전략 수립:** 유튜브 데이터 API를 활용하여 최근 30일 동안의 인기 영상을 분석하고, 다음 영상 제작에 반영할 최적의 키워드 및 형식을 찾습니다. 이를 통해 채널의 사용자 참여도를 높이고 시청자 충성도를 강화합니다.
3. **자동화 시스템 구축:** 돕는 자동화 시스템을 구축하여 더 많은 시간을 콘텐츠 제작에 집중시킬 수 있도록 합니다. 특히, AI 1인 기업의 전문성을 보여주는 대표 채널로 성장하고, 리드 유입 채널을 만들기 위한 전략을 구체적으로 계획합니다.

**현재 진행 상황:**

* **채널 분석:** 현재 채널의 활성도를 파악하고 데이터 기반으로 콘텐츠 기획 방향을 설정했습니다.
* **데이터 기반 콘텐츠 아이디어 발굴:** 최근 30일 동안 인기 영상을 분석하여, 다음 영상에 대한 키워드 및 형식 아이디어를 도출했습니다.
* **자동화 시스템 구축 (기대):** 자동화 시스템 구축을 통해 새로운 채널을 만들고 운영하는 데 집중합니다.

**다음 단계:**

1. **채널 분석 보고서 작성:** 현재 채널의 상황, 인기 콘텐츠, 시청자 반응 등을 분석하여 보고서를 작성합니다.
2. **데이터 기반 콘텐츠 기획 문서 제작:** 떡상 가능성이 높은 주제와 형식에 대한 구체적인 아이디어를 담은 기획 문서를 작성합니다.
3. **자동화 시스템 구축 계획:** 자동화 시스템의 기능, 데이터 소스, 처리 방식 등을 상세하게 설계하여 구축 계획을 세웁니다.

**필요한 도구:**

* YouTube Data API (API 키 설정)
* 유튜브 데이터 API 관련 문서 및 튜토리얼
* 자동화 시스템 구축에 필요한 기술적 지식 (Python, LLM 등)

**요청 사항:**

* **최대한 빠른 작업:** 현재 채널의 활성도를 높이고 새로운 콘텐츠를 발굴하는 데 집중하여 최대한 빨리 결과물을 도출해야 합니다.

**제안:**

*  "마스터 톤"을 유지하며, 데이터 기반 분석을 통해 유머와 진지함을 조화롭게 결합할 수 있도록 노력합니다.
*  "유튜브 챌린지"를 활용하여 새로운 콘텐츠 아이디어를 발굴하고, 시청자들의 참여를 유도합니다.

이러한 작업들을 통해 채널의 성장을 이끌어낼 수 있을 것입니다.
